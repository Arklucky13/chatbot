# Rule-Based Chatbot (Python)

This is a simple rule-based chatbot built using **if-elif-else** conditions.

## Features
- Greets the user
- Responds to basic queries
- Runs in a loop until user types `bye`

## How to Run
1. Download and extract the ZIP.
2. Open terminal in the folder.
3. Run:

```bash
python chatbot.py
```

## Files Included
- `chatbot.py` — Main chatbot script
- `README.md` — Instructions
